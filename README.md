# advent_of_code_2024

```sh
gleam run -- [number] [ --debug=<BOOL> --puzzle=<BOOL> --two=<BOOL> ]

FLAGS:
    --debug=<BOOL>        Debug an error in parsing or solving...
    --help                Print help information
    --puzzle=<BOOL>       Run on the puzzle input (not the example!)
    --two=<BOOL>          Part Two!
```

For example:

```sh
gleam run -- 1 --puzzle
```
